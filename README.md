# birds_analysis_project
# 🌿 Bird Insights Dashboard

🔗 **Live App:** https://birdsanalysisproject-hbhhmdmp8koj8wrugmuqnk.streamlit.app/

---

## 📌 Project Overview

This project analyzes bird species observation data across different habitats such as forests and grasslands. The goal is to extract meaningful insights about biodiversity, seasonal trends, and environmental impacts using data analytics techniques.

An interactive dashboard has been built using Streamlit to visualize and explore the data effectively.

---

## 🎯 Objectives

* Analyze bird species distribution across habitats
* Identify biodiversity hotspots
* Study seasonal patterns in bird activity
* Understand environmental impact (temperature, humidity)
* Provide insights for conservation and ecological research

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* SQL (MySQL)
* Streamlit
* Plotly
* Power BI

---

## 📊 Features

* 📌 Interactive dashboard with filters
* 📊 KPI metrics (Total Observations, Species Count, Avg Temperature)
* 🌿 Habitat-based biodiversity analysis
* 📅 Monthly trend visualization
* 🌦 Environmental impact analysis
* 📍 Biodiversity hotspot identification
* 🐦 Top and rare species insights

---

## 📈 Key Insights

* Forest habitats show higher biodiversity than grasslands
* Bird activity varies significantly across seasons
* Environmental factors like temperature and humidity influence behavior
* Certain locations act as biodiversity hotspots
* A few species dominate observations while others are rare
